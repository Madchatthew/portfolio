About Me

I am a web developer. I have over 20 years of experience in Computer Repair and IT Support/Administration. Currently I am a Travel Registered Nurse who has re-kindled a passion for Web Development. I started learning new programing languages, which has shown me how much I love programming. Now it is time to take that passion to the next level by becoming a full-time Web Developer. Enjoyment is found in eaither coming up with a new idea and making it come to life, or taking something already created and improving or changing the functionality.

Throughout my life, I have always loved working on websites and helping others to acheive their online goals. These goals range from being able to blog, to selling merchandise on the internet. Whether I find myself in Alaska, Virginia, South Dakota or back in Minnesota, I love to create and help indiviuals or businesses see their online ideas come to fruition.

More Background

Over that last three and a half years my family and I have had the opportunity to travel around the United States. My first Travel Nursing contract brought us to be able to spend time in two different locations in Alaska. The first being Dillingham, then Sitka. Both places were amazing and we are hoping to be able to go back to visit some day. Other Travel Nursing opportunities have provided us to travel to South Dakota where we were able to experience all of the tourists attractions, along with Custard State Park, the Needles Highway, Mount Rushmoore and the beautiful Badlands.

My family, which consists of my Wife and Daughter, were able to spend about a year in Virginia. This was the fist time I have ever been to the east coast. It was very beautiful and being able to travel to the ocean and dig up fossils along the sandy shores was amazing. We have enjoyed Travel Nursing and all of the experiences that we have experienced. We are getting ready to settle down, and with over six years in Nursing, I am looking forward to following my passion for being a Web Developer. I continue to look forward to the challenges, the successes and the never ending learning.

Theme Light Shadow
Theme extension created for a project called Flarum.io, which is a modern community forums. Flarum requires Composer to install. Node and NPM is required to created extensions. Github is used for version control.
whatsonyourscrubs.com
Whatsonyourscrubs.com was created using Flarum. It is located on a Digitalocean droplet, which is a virtual server that runs Ubuntu Server 18.04. Updates to the server and site are completed remotely. Currently this server is running Nnginx, PHP7.4 and Mariadb.
Grandforkschurch.com
Grandforkschurch.com was setup using wordpress. The theme in use was customized for appearance and functionality. Grandforkschurch.com is hosted on Siteground. Control of the site was handed off to a church member in 2017.
Nordvickarms.com
Nordvickarms includes woocommerce for the sales of products. The owner is able to use blogposts for "Latest News" section. Currently the inventory database is updated via a CSV file. API imports will be added soon. I currently update and maintain the site. Nordvickarms is hosted on Siteground.
Hollyjessen.com
Hollyjessen.com was setup using wordpress and a theme which was customized for appearance and functionality.
